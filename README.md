# Simple WeatherCard
## [Preview](https://weatherappcard.netlify.app)

### Description:
* A simple weather card using openweatherapi.
* The front of the card uses a gradient-color to reflect the temperature of the Country it is representing.
* By Clicking on the Country field the user can search for a specific country.
* Flipping the card displays the current date and time for that country.
